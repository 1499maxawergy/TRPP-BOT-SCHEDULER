time_worker module¶

Работа с временем на сервере через модуль datetime

time_worker.get_time()¶

Возврашает текущее время (MSK)

Так как вуз находится в Москве, то функция вернет время

GMT+3

time_worker.get_weekday()¶

Возвращает текущий день недели

Возвращает текущий день недели для GMT+3 на момент запроса (1 - ПН, …, 7 - ВС)

time_worker.is_even_week_of_year()¶

Возвращает четность недели

Возвращает четность неделя для GMT+3 на момент запроса (0 - четная, 1 - нечетная)
